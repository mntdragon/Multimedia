Web Sockets

Read the 'Introducing WebSockets: Bringing Sockets to the Web' article.
What kind of problems web sockets solve? In what kind of situations you should use them?

Web sockets support the 2 way communicated connection between client and server more efficient and less cost consuming. They are easy to apply in many kinds of applications. Moreover, they help saving bandwodth, CPU and reduce the latency Client-Server and Server-Client connections, easy to solve the errors. We can use Web Sockets with node.js ans Socket.io (JavaScript library). When making 2 way connection between client and server, we will need the great advantages of Web Sockets.
